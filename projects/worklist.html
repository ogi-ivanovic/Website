About Project

Web app developed with Python and Flask which allows users to create a profile containing information such as professional skills, projects and experience. The app then uses the user's information to find jobs which best fit them based on their skills, projects and experience. When the user is applying to a job, the app uses a machine learning algorithm to generate a resume from the user's information which best fits the job that the user is applying to. The user is able to tweak the generated resume and apply quickly and easily to the job.
