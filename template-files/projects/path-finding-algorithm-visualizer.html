About Project

This project allows the user to set a start and end point on a grid and then displays a search algorithm in real-time from the start point to the end point. You are able to choose which search algorithm you want to display and you are able to draw walls on the grid. Currently, I am in the process of adding mutiple different search algorithms to be displayed such as Dijkstra's algorithm and A* algorithm.
